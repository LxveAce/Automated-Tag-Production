# Security Policy

## Supported versions

Only the latest commit on the default branch is actively maintained. No backported security fixes are made to older commits or releases.

## Reporting a vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

- **Email:** lxveace@proton.me
- **Subject line:** `[SECURITY] Automated-Tag-Production — <brief description>`

Please include:

1. A description of the vulnerability and its potential impact
2. Steps to reproduce or a proof-of-concept (if applicable)
3. Any suggested mitigations you have in mind

Do **not** open a public GitHub issue for security vulnerabilities.

You can expect an acknowledgement within a few business days. This is a solo open-source project; response times may vary.
